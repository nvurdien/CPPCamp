Code Campers C++ resources:

Here are some of the resources that the Code Camper crew found especially useful when scouring the internet for C++ knowledge.

cplusplus

Cplusplus.com is a great resource for finding information on the standard libraries and general C++ language. They also have a few introductory tutorials that can be useful for new students.

Tutorialspoint

Tutorialspoint.com has some of our favorite introductory lessons to much of the same topics that we cover in our lessons. If you are struggling with concepts we recommend you see if Tutorialspoint has any helpful material.

Khan Academy

Khan Academy is recognized as one of the greats when it comes to online learning. Khan Academy does not disappoint when explaining general computer science topics, but does not go into the specifics of the C++ language. Still, we recommend you use their videos to get general Computer Science background knowledge.

Cprogramming.com

Cprogramming.com is a great resource if you want something much more in-depth than the lessons that we offer. The downside for this is that many students get bored and tired after a single lesson, but nonetheless it is quality information.
